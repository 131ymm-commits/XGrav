# XGrav

**UAF-Powered Evolutionary Gravity Laboratory**

Проект, где физика, эволюционные алгоритмы и UAF-когерентность используются для поиска **красивых, стабильных и реалистичных** гравитационных систем — астероидных облаков, колец с пастухами и резонансных структур.

---

### ✨ Основные возможности

- UAF Γ-когерентность как главная метрика стабильности
- Поддержка орбитальных резонансов
- Shepherd Moons (пастухи колец)
- Долгосрочная интеграция (до t=7000+)
- Генерация эстетически привлекательных конфигураций
